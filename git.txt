                 一 -----  git的基本操作
1.1  创建一个文件，在文件里面的打开git here 并输入 git init会出现一个.git的文件
1.2  git分为工作区，暂存区，本地仓库，github仓库
	工作区就是自己写的代码
1.3  工作区提交到暂存区
	git add . 提交所有修改的代码（文件）
	git add 文件名
1.4  暂存区提交到本地仓库
	git commit -m '提交的说明'
1.5  回复上一次操作
	git reset --hard 标识
1.6  查看状态
      	git status ）（红色的是在工作区，绿色的是在暂存区）
1.7  查看历史版本
	git log
1.8  查看版本标识（以一行展示）
	git reflog
